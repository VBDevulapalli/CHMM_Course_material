## 🔥 FLAMMABILITY QUICK REFERENCE

### Flash Point Classifications
```
Flash Point < 100°F (37.8°C)    → FLAMMABLE LIQUID (OSHA)
Flash Point 100-200°F            → COMBUSTIBLE LIQUID (OSHA)
Flash Point > 200°F              → NOT REGULATED AS FLAMMABLE
Flash Point < 60°C (140°F)       → FLAMMABLE/COMBUSTIBLE (DOT)
```

### Common Chemicals Flash Points
| Chemical | Flash Point (°C) | Classification |
|----------|------------------|-----------------|
| Gasoline | -43 | Extremely flammable |
| Acetone | -20 | Highly flammable |
| Benzene | -11 | Highly flammable |
| Ethanol | 13 | Highly flammable |
| Toluene | 4 | Highly flammable |
| Diesel fuel | 52 | Flammable |
| Mineral oil | 160+ | Non-flammable |
| Water | N/A | Non-flammable |

#### Acute Toxicity - Oral LD50
| Category | LD50 Range | Hazard |
|----------|-----------|--------|
| Cat 1 | ≤ 5 mg/kg | Fatal |
| Cat 2 | 5-50 mg/kg | Fatal |
| Cat 3 | 50-300 mg/kg | Toxic |
| Cat 4 | 300-2000 mg/kg | Harmful |
| Cat 5 | 2000-5000 mg/kg | May be harmful |

### Exposure Limit Hierarchy
```
HIGHEST PROTECTION:
TLV-C (Ceiling, never exceed) 
  ↓
TLV-STEL (15-min average)
  ↓
TLV-TWA (8-hour average)
  ↓
PEL (OSHA legal limit, often less protective than TLV)
  ↓
REL (NIOSH recommended limit)

**Pattern:** ACGIH TLVs are usually MORE protective (lower) than OSHA PELs
```

## 🚚 DOT HAZARD CLASS QUICK REFERENCE

### All 9 Classes at a Glance

```
CLASS 1: EXPLOSIVES
├─ 1.1: Mass explosion (TNT)
├─ 1.2: Projection hazard (ammunition)
├─ 1.3: Fire & minor blast (fireworks)
├─ 1.4: Minor hazard (consumer fireworks)
├─ 1.5: Blasting agents (ANFO)
└─ 1.6: Extremely insensitive (detonators)

CLASS 2: GASES
├─ 2.1: Flammable (propane, butane)
├─ 2.2: Non-flammable/non-toxic (nitrogen, oxygen)
└─ 2.3: Toxic (chlorine, hydrogen sulfide)

CLASS 3: FLAMMABLE/COMBUSTIBLE LIQUIDS
└─ (No divisions) Flash point <60°C (140°F)

CLASS 4: FLAMMABLE SOLIDS
├─ 4.1: Flammable solid (red phosphorus)
├─ 4.2: Spontaneously combustible (white phosphorus)
└─ 4.3: Dangerous when wet (sodium, calcium carbide)

CLASS 5: OXIDIZERS & ORGANIC PEROXIDES
├─ 5.1: Oxidizers (peroxides, permanganates)
└─ 5.2: Organic peroxides (benzoyl peroxide)

CLASS 6: TOXIC & INFECTIOUS
├─ 6.1: Toxic (pesticides, cyanides)
└─ 6.2: Infectious (biohazards, medical waste)

CLASS 7: RADIOACTIVE MATERIAL
└─ (No divisions) Alpha, beta, gamma emitters

CLASS 8: CORROSIVE
└─ (No divisions) Acids, bases, oxidizers causing corrosion

CLASS 9: MISCELLANEOUS
└─ (No divisions) Other hazardous materials
```

## 🧬 RCRA CHARACTERISTICS (IRTC) QUICK REFERENCE

### Ignitability (D001)
```
TRIGGERS FOR D001:
✓ Liquid with flash point < 60°C
✓ Non-liquid that ignites by friction/moisture
✓ Ignitable compressed gas
✓ Oxidizer (per DOT definition)

TEST: Typically measured by Pensky-Martens or Tag closed-cup method
THRESHOLD: Any amount = hazardous
DISPOSAL: Incineration, fuel blending (energy recovery)
```

### Corrosivity (D002)
```
TRIGGERS FOR D002:
✓ Aqueous liquid with pH ≤ 2.0 OR pH ≥ 12.5
✓ Liquid corroding steel >6.35 mm/year at 55°C

TEST: pH paper or meter (for pH); steel coupon immersion (for corrosion rate)
THRESHOLD: Any amount = hazardous
STORAGE: Separate from bases/acids; corrosion-resistant containers
DISPOSAL: Neutralization, treatment
```

### Reactivity (D003)
```
TRIGGERS FOR D003:
✓ Normally unstable, reacts violently
✓ Reacts violently with water
✓ Forms explosive with water
✓ Generates toxic gas with water
✓ Contains cyanide/sulfide (toxic gas at pH 2-12.5)
✓ Capable of detonation
✓ Listed as explosive by DOT

EXAMPLES: Sodium metal, white phosphorus, calcium carbide, cyanide salts
DISPOSAL: Special handling; incineration with acid gas scrubbing
```

### Toxicity (D004-D043)
```
TEST METHOD: TCLP (Toxicity Characteristic Leaching Procedure)
WHY TCLP?: Simulates waste in landfill; determines if leachate exceeds limits
REGULATED CHEMICALS: 40 specific substances (metals, VOCs, pesticides)

IF TCLP RESULT EXCEEDS LIMIT → WASTE CODE D0XX ASSIGNED
EXAMPLE: Soil with arsenic leachate = 8 mg/L → Exceeds 5 mg/L limit → D004
```

## 🏷️ GHS PICTOGRAMS & SIGNAL WORDS

### Quick Pictogram ID Chart

| # | Pictogram | Hazard Class | Signal Word | Examples |
|---|-----------|--------------|-------------|----------|
| 1 | 🔥 Flame | Flammable (solid/liquid/gas) | DANGER | Gasoline, propane |
| 2 | 🔥⭕ Flame/Circle | Oxidizing | DANGER | Bleach, hydrogen peroxide |
| 3 | 💣 Bomb | Explosive/Self-reactive | DANGER | Fireworks, TNT |
| 4 | ☠️ Skull/Bones | Acute Toxicity (fatal/toxic) | DANGER | Cyanide, arsenic |
| 5 | ⚠️ Exclamation | Irritant/Sensitizer/Harmful | WARNING | Detergents, solvents |
| 6 | 🏥 Health | Carcinogen/Mutagen/Repro | DANGER | Asbestos, benzene |
| 7 | 🧪 Corrosion | Skin corrosion/eye damage | DANGER | Sulfuric acid, NaOH |
| 8 | 🔴 Gas Cylinder | Gas under pressure | WARNING | Propane, nitrogen |
| 9 | 🌍 Environment | Aquatic toxicity | WARNING | Pesticides (NOT OSHA required) |

### Signal Word Decision Tree
```
Is it FATAL or SEVERELY TOXIC from single/short exposure?
    → DANGER
Is it a CARCINOGEN, MUTAGEN, or REPRODUCTIVE TOXICANT?
    → DANGER
Is it MODERATELY HAZARDOUS (e.g., irritant)?
    → WARNING
Otherwise → Check GHS guidance for specific hazard
```

## 📚 NFPA 704 QUICK REFERENCE

### Rating Scale (0-4) for Each Quadrant
```
        ╔════════╗
        ║   RED  ║  FLAMMABILITY
        ║  0: Non-flammable
        ║  1: Flash > 200°F (93°C)
        ║  2: Flash 100-200°F
        ║  3: Flash < 100°F
        ║  4: Flash < 73°F or gas
╠═══════╩════════╣
║ BLUE           ║ YELLOW
║ HEALTH   REACTIVITY
║ 0: Normal     0: Stable
║ 1: Slight     1: Unstable if heated
║ 2: Moderate   2: Can explode if heated
║ 3: Serious    3: Can detonate
║ 4: Deadly     4: Will detonate
║               
╚═══════════════╝
        ║        ║
      WHITE
    SPECIAL HAZARDS
    (only note if present)
    W̄ = Water reactive
    OX = Oxidizer
    SA = Simple asphyxiant
```

### Example NFPA 704 Reading
```
      ┌─┐
      │3│ RED (Flash point < 100°F — highly flammable)
  ┌─┬─┴─┴─┬─┐
  │1│     │0│ BLUE=1 (slight health hazard)
  └─┴─────┴─┘     YELLOW=0 (stable)
        │ │
        │OX│ = Contains oxidizer
        └─┘

INTERPRETATION: This chemical is flammable with a slightly hazardous
health profile and contains an oxidizer. Store separately from other
flammables and incompatible materials.
```

## 🌍 ENVIRONMENTAL FATE QUICK REFERENCE

### Bioaccumulation Risk Assessment
```
Log Kow VALUE:
log Kow > 4      → HIGH bioaccumulation potential (e.g., DDT, PCBs)
log Kow 2-4      → MODERATE bioaccumulation potential
log Kow < 2      → LOW bioaccumulation potential

**Historical Examples:**
DDT (log Kow ~6.9)          → Bioaccumulated 8 million times → banned
PCBs (log Kow ~6-8)         → Bioaccumulated → banned
Mercury (log Kow varies)    → Methylmercury bioaccumulates in fish
Benzene (log Kow ~2)        → Low bioaccumulation
```

### Environmental Persistence
```
HIGHLY PERSISTENT (Years to centuries):
✗ PCBs, dioxins, furans, DDT
✗ Heavy metals (especially mercury)
✗ Some industrial chemicals

MODERATELY PERSISTENT (Months to years):
⚠️ Some pesticides
⚠️ Certain volatile organics

RAPIDLY DEGRADABLE (Days to weeks):
✓ Alcohols, simple sugars
✓ Some herbicides
✓ Readily biodegradable compounds
```

## 💾 KEY DEFINITIONS — MUST MEMORIZE

```
HAZARDOUS MATERIAL (DOT):     Substance capable of posing unreasonable
                               risk during transportation

HAZARDOUS WASTE (EPA/RCRA):   Discarded material meeting EPA criteria
                               (on list OR exhibits IRTC characteristic)

HAZARDOUS CHEMICAL (OSHA):    Chemical classified by GHS as hazardous
                               (health or physical hazard)

REPORTABLE QUANTITY (RQ):     Minimum amount triggering spill reporting
                               to National Response Center (1-800-424-8802)

THRESHOLD QUANTITY (TQ):      Minimum stored amount triggering EPCRA
                               Section 312 reporting

IDLH:                         Max concentration allowing 30-min safe
                               escape without permanent injury (NIOSH)

DOSE-RESPONSE:                Relationship between exposure amount and
                               biological effect ("dose makes the poison")

TCLP:                         Lab test simulating landfill leaching
                               to determine waste toxicity

MULTIMEDIA CONTAMINANT:       Chemical found in water, soil, and air
                               (e.g., benzene, toluene, xylene)

BIOACCUMULATION:              Chemical buildup in organism >
                               elimination rate

BIOMAGNIFICATION:             Concentration increase up food chain
```

## ❌ TOP INCOMPATIBILITIES FOR STORAGE

```
RULE 1: OXIDIZERS vs. FLAMMABLES
        ✗ Never store together
        ✗ Examples: H₂O₂ + gasoline = fire risk

RULE 2: ACIDS vs. BASES
        ✗ Heat & violent reaction
        ✗ Examples: HCl + NaOH = heat release

RULE 3: ACIDS vs. CYANIDES
        ✗ Hydrogen cyanide gas (TOXIC)
        ✗ Example: H₂SO₄ + NaCN = HCN gas

RULE 4: ACIDS vs. SULFIDES
        ✗ Hydrogen sulfide gas (TOXIC)
        ✗ Example: HCl + FeS = H₂S gas

RULE 5: CHLORINE BLEACH vs. AMMONIA
        ✗ Chloramine gas (TOXIC)
        ✗ Used as weapon; extremely hazardous

RULE 6: WATER-REACTIVE + WATER
        ✗ Violent reaction, flammable/toxic gases
        ✗ Examples: Na metal + H₂O, CaC₂ + H₂O
```

## 🎯 EXAM PATTERN RECOGNITION

### Pattern 1: "Worker in basement exposed to gas"
→ Think HEAVIER-THAN-AIR GASES (vapor density > 1)
→ Chlorine, propane, H₂S, SO₂ likely
→ SCBA probably required

### Pattern 2: "Spill requires cleanup"
→ Check for WATER-REACTIVE materials (can't use water-based methods)
→ Check for ACID/BASE incompatibilities (neutralization = heat)
→ Look at TCLP limits for contaminated soil

### Pattern 3: "Chemical storage inspection"
→ Look for OXIDIZER/FLAMMABLE violations
→ Look for ACID/BASE incompatibilities
→ Check WATER-REACTIVE materials in moist environments

### Pattern 4: "Lab test shows contamination"
→ Compare to TCLP LIMITS
→ Determine RCRA WASTE CODE (D001-D043)
→ Identify disposal/treatment requirements

### Pattern 5: "Transportation of chemicals"
→ Determine DOT HAZARD CLASS
→ Check for DOT PACKING GROUP
→ Verify proper SHIPPING PAPERS, LABELING, PLACARDING

### Pattern 6: "SDS Review"
→ Section 8 = PPE & exposure controls
→ Section 9 = Flash point & fire hazard assessment
→ Section 11 = Toxicological route of exposure
→ Section 14 = DOT classification for transport
