## 🗂️ About This Course

This repository contains structured daily study materials for the **CHMM (Certified Hazardous Materials Manager)** exam. Each day's folder includes:

- **Complete Notes** — In-depth coverage of the day's topic
- **Quick Reference** — Condensed key facts for last-minute review
- **Practice Questions** — Multiple-choice questions with detailed explanations (collapsible answers)

## 🚀 How to Get Started

1. Open the **Quick Navigation** table above
2. Click the day you want to study
3. Start with **Complete Notes**, then **Quick Reference**, then test yourself with **Practice Questions**

## 📋 About the CHMM Exam

| Detail | Info |
|--------|------|
| **Certifying Body** | Institute of Hazardous Materials Management (IHMM) |
| **Number of Questions** | 185 multiple-choice |
| **Time Allowed** | 3.5 hours |
| **Passing Score** | ~70% |
| **Exam Domains** | Regulations, Environmental, Safety, Emergency Response, Risk |
| **Eligibility** | Bachelor's degree + 3 years hazmat experience (or equivalent) |

## 📝 Prerequisites

- Basic understanding of chemistry and environmental science
- Familiarity with U.S. federal regulatory framework
- Access to 49 CFR, 40 CFR, and 29 CFR (linked above)

---

## 💡 Study Tips for Success

- Review **Complete Notes** first, then reinforce with **Quick Reference**
- Take **Practice Questions** without looking at answers first
- Focus on regulatory agency jurisdiction — exam loves to test which agency covers what
- Memorize key CFR titles: DOT = 49, EPA = 40, OSHA = 29, NRC = 10
