## Study Tips for Success

1. **Form study groups** — Teach concepts to others
2. **Use active recall** — Test yourself frequently
3. **Create concept maps** — Link related topics across domains
4. **Review daily memorization lists** — Spaced repetition is key
5. **Track weak areas** — Focus extra time on challenging domains
6. **Time management** — Aim for 45-minute focused study blocks with 10-minute breaks

---

## About the CHMM Exam

- **Format:** Multiple choice (120 questions)
- **Duration:** 4 hours
- **Passing Score:** 70%
- **Domains Tested:** 12 (as shown in table above)
- **Prerequisites:** Minimum 3 years of full-time experience in hazmat management

For more info: https://www.ihmm.org/chmm-exam/
